## Sobre o projeto

O projeto **Barão Zagottis Bar e Restaurante** foi desenvolvido com a proposta de criar uma **solução web para um contexto real**, simulando uma demanda profissional de um comércio local. O objetivo principal foi desenvolver um site institucional para fortalecer a presença digital do estabelecimento, melhorar a divulgação dos serviços e facilitar o contato com o público.

---

## Contexto escolhido

O contexto escolhido pelo grupo foi o de um **comércio local do ramo de alimentação e entretenimento**, mais especificamente um **bar e restaurante**.

Esse tipo de negócio depende fortemente de:
- visibilidade online;
- divulgação de produtos e serviços;
- comunicação rápida com clientes;
- apresentação clara de horários, localização e diferenciais do local.

---

## Problema identificado

Antes do desenvolvimento do projeto, identificamos algumas necessidades comuns a esse tipo de estabelecimento:

- ausência ou limitação de presença digital organizada;
- dificuldade para divulgar os drinks, eventos e diferenciais do restaurante;
- pouco destaque para informações importantes, como:
  - cardápio de drinks;
  - localização;
  - horários de funcionamento;
  - contato via WhatsApp;
  - transmissões esportivas e eventos.
